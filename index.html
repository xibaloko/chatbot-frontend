O que é
Arcade Note

Arcade Note é um sistema inteligente capaz de manter uma conversa com o seu usuário, reconhecendo as suas perguntas e traz as informações que você deseja. As informações são extraídas de uma ampla gama de dicionários e cresce dia após dia. Saiba tudo sobre o seu jogo Favorito. Uma pergunta, uma resposta, um problema resolvido.

O que é um ChatBot

Chatbot é um programa de computador que tenta simular o diálogo com um ser humano. O objetivo é responder as perguntas de tal forma que as pessoas tenham a impressão de estar conversando com outra pessoa e não com um robo.

Como o Arcade Note sabe o que me recomendar

Basicamente, o software registra todas as palavras escritas pelo usuário para processamento posterior. Dessa forma, ao invés de dar respostas pré-programadas, o bot busca por palavras relacionadas e responde com sugestões adequadas sobre o assunto.

Qual é o critério para recomendação de jogos

De acordo com as suas pesquisas, o Arcade note identificara o estilo de jogo que você gosta e trará opções relacionadas, com o objetivo de te trazer novidades sobre o universo gamer.

De onde são tiradas as informações dos jogos

Assim que feito uma pergunta, nossa ferramenta de busca traz tudo sobre a assunto desejado, de uma forma formatada para que o usuário tenha uma ótima experiência.
